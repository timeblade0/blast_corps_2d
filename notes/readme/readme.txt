Controls:
----------------------------
Left/Right = Rotate Vehical
Up = Accelerate
Down = Reverse
Space = Exit Vehicle
Enter = Return to Stage Select
Esc = Exit Game
F2 = Restart Game

Notes:
---------------------------------------
*to enter a vehicle, walk into it
*when you exit a vehicle, you exit below it, except for trains which exits left
*the grey lights are called RDU's. If you run over them, they light up. If you get them all, you get a medal.
*You get a medal for: beating the level, getting all RDU's, and destroying all buildings.
*The small satilite are Com Links. Touching one will give you a bonus level.
